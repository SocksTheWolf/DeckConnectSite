Hi everyone! I hope the new year is treating you well.

Recently, Elgato has announced that they will require DRM on all Stream Deck plugins in order to push updates.

To counteract this change and allow other platforms (such as Linux) to continue being able to use DeckConnect, I have provided an [alternative download link](/deck-direct) on the download page that will allow you to always get the very latest version of DeckConnect DRM free.

As a reminder, the SD plugin will always be free to download.

I've also pushed up a new version to the Elgato Marketplace with some dependency updates to keep things running smoothly and securely.

I'll make another post when that is approved and published.

Thanks for using DeckConnect!
